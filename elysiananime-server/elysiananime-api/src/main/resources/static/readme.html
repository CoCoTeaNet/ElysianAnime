ElysianAnime

项目简介

这是一个番剧管理系统，主要用于部署在个人NAS服务器上，订阅和下载RSS内容，然后自动追番~~~

  • version: v2.2.0
  • Changed Log

图片演示

功能要点

  • 蜜柑RSS订阅自动下载内容
  • 在线番剧检索和播放
  • 番剧信息管理
  • 追番进度管理
  • 番剧推荐栏目

启动说明

步骤

  1. 运行数据库脚本
  2. 启动本地Redis并运行后端服务
  3. 安装前端依赖并运行前端项目

访问

  • 地址：http://localhost:8086
  • 账号：admin admin123456

环境要求

  • JDK：17+
  • Nodejs：14+
  • MySQL：8.0+
  • Redis
  • qBittorrent

拓展项目

  • ElysianAnime-Viewer 桌面端应用（嵌入Mpv,播放体验更好）
