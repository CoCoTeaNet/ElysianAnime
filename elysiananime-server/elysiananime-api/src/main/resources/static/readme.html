🎬 ElysianAnime

私人服务器上的动漫追番神器

GitHub 项目地址

🌟 简介

ElysianAnime 是一个专为动漫爱好者打造的自动化追番管理系统。你可以：

  • 📺 自动订阅并下载 RSS 中的番剧资源
  • 🧩 部署在你的私人服务器上，完全掌控数据
  • 🎞️ 搭配桌面端播放器 ElysianAnime-Viewer，享受内嵌 MPV 的高清播放体验

🛠 技术栈 & 运行环境

  • JDK 17+
  • MySQL 8.0+
  • Redis 支持
  • Node.js 14+
  • qBittorrent（推荐）
  • 前端：Vue / React 可选

🚀 快速开始

  1. git clone https://github.com/CoCoTeaNet/ElysianAnime.git
  2. 运行数据库脚本初始化表结构
  3. 启动 Spring Boot 后端服务
  4. 安装前端依赖并运行：
    npm install
    npm run serve
  5. 访问：http://localhost:8086（默认账号：admin/admin123456）
