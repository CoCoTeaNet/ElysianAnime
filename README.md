# JAnime

## 项目简介

这是一个番剧管理系统，主要用于部署在个人NAS服务器上，订阅和下载RSS内容，然后自动追番~~~

- version: v2.0.8


## 功能要点

- [X] 蜜柑RSS订阅自动下载内容
- [X] 在线番剧检索和播放
- [X] 番剧信息管理
- [X] 追番进度管理
- [X] 番剧推荐栏目


## 启动说明
### 步骤
1. 运行数据库脚本
2. 启动本地Redis并运行后端服务
3. 安装前端依赖并运行前端项目

* 访问地址
* http://localhost:8086


## 环境要求
- JDK：1.8+
- Nodejs：14 ~ 19
- MySQL：8.0+
- Redis
- qBittorrent
